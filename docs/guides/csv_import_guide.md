# CSV一括登録マニュアル

> この機能は **Teamプラン** 専用です。

## 概要

CSV一括登録を使うと、複数のサロンやスタイリストの情報をまとめて登録できます。大規模サロンチェーンで多数の店舗・スタイリストを効率的にセットアップする際に便利です。

## CSVフォーマット仕様

### サロン用CSV

| カラム名 | 必須 | 最大文字数 | 説明 |
|---------|------|-----------|------|
| `name` | 必須 | 100 | サロン名 |
| `area` | 必須 | 100 | エリア（例: 渋谷、表参道） |
| `concept` | 任意 | 500 | サロンコンセプト |
| `target_customer` | 任意 | 200 | ターゲット層 |
| `strength` | 任意 | 500 | 強み・特徴 |

**サンプル:**
```csv
name,area,concept,target_customer,strength
渋谷サロンA,渋谷,ナチュラル美容,20代女性,透明感カラー
表参道サロンB,表参道,大人エレガント,30代女性,ハイライト・バレイヤージュ
```

### スタイリスト用CSV

| カラム名 | 必須 | 説明 | バリデーション |
|---------|------|------|-------------|
| `salon_name` | 必須 | 所属サロン名（組織内に存在するサロン名と完全一致） | - |
| `name` | 必須 | スタイリスト名 | 最大100文字 |
| `role` | 任意 | 役職（店長、スタイリスト等） | 最大50文字 |
| `years_experience` | 任意 | 経験年数 | 0〜50の整数 |
| `specialties` | 任意 | 得意技術 | セミコロン`;`区切り |
| `style_features` | 任意 | スタイル特徴 | セミコロン`;`区切り |
| `personality` | 任意 | 性格・人柄 | 最大200文字 |
| `writing_tone` | 任意 | 文体トーン | casual / formal / friendly / professional |
| `writing_emoji` | 任意 | 絵文字使用度 | none / minimal / moderate / frequent |
| `writing_sentence_style` | 任意 | 文の長さ | short / medium / long |

**サンプル:**
```csv
salon_name,name,role,years_experience,specialties,style_features,personality,writing_tone,writing_emoji,writing_sentence_style
渋谷サロンA,田中太郎,店長,15,カット;カラー;パーマ,ナチュラル;エアリー,明るくて優しい,friendly,moderate,medium
渋谷サロンA,佐藤花子,スタイリスト,8,カラー;トリートメント,透明感;ツヤ,穏やかで丁寧,professional,minimal,long
```

## 配列フィールドのセミコロン区切り

`specialties` と `style_features` は **セミコロン `;`** で区切ります。

- 正しい例: `カット;カラー;パーマ`
- 間違い: `カット,カラー,パーマ`（カンマで区切るとCSVのカラムと混同します）

## writing_style の選択肢一覧

### writing_tone（文体トーン）
| 値 | 説明 |
|----|------|
| `casual` | カジュアルなトーン |
| `formal` | フォーマルなトーン |
| `friendly` | 親しみやすいトーン（デフォルト） |
| `professional` | プロフェッショナルなトーン |

### writing_emoji（絵文字使用度）
| 値 | 説明 |
|----|------|
| `none` | 絵文字なし |
| `minimal` | 控えめ（デフォルト） |
| `moderate` | 適度に使用 |
| `frequent` | 頻繁に使用 |

### writing_sentence_style（文の長さ）
| 値 | 説明 |
|----|------|
| `short` | 短い文 |
| `medium` | 標準的な長さ（デフォルト） |
| `long` | 長い文 |

省略した場合はデフォルト値が適用されます。

## 操作手順

### 1. テンプレートをダウンロード

CSV一括登録ページ（`/team/import`）から、サロン用またはスタイリスト用のCSVテンプレートをダウンロードします。

### 2. データを記入

ダウンロードしたテンプレートに、登録したいデータを記入します。

- **エンコーディング**: UTF-8（BOM付きも対応）
- **ファイルサイズ上限**: 5MB
- **最大行数**: 1,000行

### 3. アップロード

CSVファイルをドラッグ&ドロップ、またはファイル選択でアップロードします。

### 4. プレビュー確認

アップロード前に先頭5行のプレビューが表示されます。内容を確認してください。

### 5. インポート実行

「インポート実行」ボタンをクリックすると、バリデーションとデータ登録が行われます。

### 6. 結果確認

- **成功**: 登録件数が表示されます
- **失敗**: エラーの行番号・フィールド・理由が一覧表示されます

## エラー対処法

### よくあるエラーと解決方法

| エラー | 原因 | 解決方法 |
|-------|------|---------|
| 必須カラムが見つかりません | ヘッダー行にname/areaが無い | CSVの1行目にヘッダーを正しく記入 |
| サロン「○○」が組織内に見つかりません | スタイリストCSVのsalon_nameが一致しない | 先にサロンを登録するか、サロン名を正確に記入 |
| ファイルサイズが5MBを超えています | ファイルが大きすぎる | データを分割してアップロード |
| 行数が上限を超えています | 1,000行を超えている | データを分割してアップロード |
| writing_toneは... | 無効な値 | 選択肢一覧から正しい値を選択 |

### 重要: All or Nothing

CSV一括登録は **All or Nothing** 方式です。1行でもバリデーションエラーがあると、全行がロールバックされます。エラーを修正してから再度アップロードしてください。
