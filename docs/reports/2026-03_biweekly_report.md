## 3. 本番デプロイで解決した問題

3/18〜19の本番投入で遭遇・解決した問題の一覧:

| # | 問題 | 原因 | 解決策 |
|---|------|------|--------|
| 1 | Railway 502 Bad Gateway | PORT環境変数未使用（8000固定） | `$PORT` を動的に取得 |
| 2 | サロン登録 "Failed to fetch" | フロントのAPIベースURLがlocalhost固定 | 環境変数に切り替え |
| 3 | CORS エラー | Vercelドメインが未許可 | `allow_origins` に追加 |
| 4 | 全API 500エラー | supabase 2.3.4 と httpx<0.26 の組み合わせで gotrue が `proxy` 引数エラー | supabase 2.11.0 + httpx 0.28+ にアップグレード |
| 5 | DB接続 Circuit Breaker | DATABASE_URLのパスワードが未設定（プレースホルダーのまま） | Supabaseでパスワードリセット＆設定 |
| 6 | SalonResponse ValidationError | PostgreSQLがUUIDオブジェクトを返すがPydanticはstr期待 | 全モデルの `to_dict()` で `str()` 変換 |
| 7 | ブログ生成 500エラー | `subscriptions.trial_end` カラムがDBに未作成 | マイグレーションSQL作成（005_missing_columns.sql） |
