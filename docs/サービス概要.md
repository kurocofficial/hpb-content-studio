# HPB Content Studio — サービス概要

## サービスコンセプト

**「ブラウザのタブひとつで、HPBの文章運用がすべて完結する世界」**

HPB Content Studioは、ホットペッパービューティー（HPB）向けテキストコンテンツをAIで効率的に生成するWebアプリケーションです。美容室の集客担当スタッフが、スタイリストの個性に合わせたブログ記事・紹介文・プロフィール等を短時間で作成できます。

---

## 解決する課題

美容室がHPBに掲載する際、以下のような大量のテキストコンテンツを継続的に作成・更新する必要があります。

- サロン紹介文・キャッチコピー
- スタイリストプロフィール
- ブログ記事（毎週更新）
- 口コミ返信
- クーポン・メニュー説明文

集客担当スタッフにとって、**複数スタイリストの個性に合わせたコンテンツの定期作成**は大きな時間的負担です。結果として、ブログが放置されたり紹介文が長期間更新されないサロンが多数存在しています。

---

## ターゲットユーザー

**美容室の集客担当スタッフ**

- HPB管理画面（サロンボード）の操作経験あり
- 文章作成に苦手意識を持つ層が多い
- ITリテラシーは中程度（スマホネイティブ、PC操作は基本レベル）
- 20〜30代女性が中心

## 主な機能

### テキストコンテンツ生成（AI: Claude 4.5 Haiku）

スタイリストのメタデータ（得意メニュー・文体の好み・人柄など）を自動的にプロンプトに合成し、「その人らしい」文章をAIが生成します。

| コンテンツ種別 | HPB文字数上限 | 状態 |
|--------------|-------------|------|
| サロンキャッチコピー | 45文字 | ✅ 実装済み |
| サロン紹介文 | 500文字 | ✅ 実装済み |
| スタイリストプロフィール | 200文字 | ✅ 実装済み |
| ブログ記事 | 10,000文字 | ✅ 実装済み |
| 口コミ返信 | 500文字 | ✅ 実装済み |
| カウンセリング文 | 2,000文字 | ✅ 実装済み |
| Google口コミ返信 | 500文字（推奨） | ✅ 実装済み |
| クーポン説明文 | 200文字 | Phase 2 |
| メニュー説明文 | — | Phase 2 |

### チャット修正

生成したコンテンツをチャットUIで対話的に修正できます。「もう少し大人っぽく」「絵文字を減らして」などの自然な指示でAIが即座にリライトします。

### コピペ最適化

HPBへのコンテンツ反映はコピー＆ペースト方式です（HPBは公開APIを提供していないため）。ワンクリックコピーボタン＋HPB基準のリアルタイム文字数カウンターで、スムーズな転記体験を実現します。

### スタイリスト管理

フォーム形式でスタイリスト情報を登録。以下のメタデータが全コンテンツ生成に自動反映されます。

- 得意メニュー（カット、カラー、パーマ等）
- スタイルの特徴（ナチュラル、透明感、韓国風等）
- 人柄・雰囲気
- 文体の好み（フレンドリー / プロフェッショナル / トレンド感）

### Teamプラン機能

大規模美容サロンチェーン向けのプランです。

- **マルチサロン管理**: 複数店舗を一元管理
- **CSV一括登録**: サロン・スタイリスト情報のCSVインポート
- **メンバー管理**: オーナー / 管理者 / メンバーの権限管理
- **管理者CLIツール**: コマンドラインからの運用管理

## システムアーキテクチャ

```
ブラウザ (React + Vite)
    │  HTTPS
    ▼
FastAPI バックエンド
    │
    ├── Prompt Engine（メタデータ + テンプレート + HPB制約 → プロンプト合成）
    ├── Claude API（テキスト生成 / SSEストリーミング）
    ├── Supabase Auth（認証）
    ├── Supabase PostgreSQL + RLS（データ保存・分離）
    └── Stripe API（決済 / Phase 2）
```

**プロンプトエンジンの仕組み**: スタイリストのメタデータ・サロン情報・HPB文字数制約・コンテンツ固有パラメータを自動合成し、Claude APIに送信。生成時のプロンプトはDBに保存され、再現性を担保します。

## 想定利用シーン

1. **毎週のブログ更新** — テーマを入力するだけで、スタイリストの文体でブログ記事を生成
2. **新スタイリスト入社** — プロフィール一式をまとめて作成
3. **季節の一括更新** — 紹介文やスタイル説明を季節に合わせて一括リフレッシュ
4. **口コミ返信** — お客様の口コミを貼り付けるだけで、適切なトーンの返信文を生成
5. **Google口コミ対応** — Googleビジネスプロフィールの口コミにも対応
